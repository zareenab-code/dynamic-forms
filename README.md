## Tech Stack

-   Laravel
-   MySQL
-   Bootstrap

## Project Setup Guidelines

-   Clone repository
-   cd into cloned repository
-   Run `composer update`
-   Run `npm install`
-   create env file
-   Add database details in env file
-   Run `php artisan key:generate`
-   Run `php artisan migrate`
-   Run `php artisan db:seed`
-   Run `php artisan serve`
